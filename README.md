
  # Sistema de Control de Cumplimiento

  This is a code bundle for Sistema de Control de Cumplimiento. The original project is available at https://www.figma.com/design/8GHkGWCGeym0uxYkVgoBI1/Sistema-de-Control-de-Cumplimiento.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run electron:dev` to start the app in Electron with Vite.

  Run `npm run dev:desktop` to start the desktop app in development mode.

  Run `npm run electron` to build and open the production Electron app.
